## O aplikacji

**Godai - Karta Postaci** to darmowa aplikacja dla systemu Windows przeznaczona do obsługi postaci w systemie **Godai RPG**.

Aplikacja umożliwia szybkie i intuicyjne tworzenie postaci, automatyzując wiele elementów mechaniki systemu. Projekt został przygotowany zarówno z myślą o nowych graczach, jak i doświadczonych użytkownikach Godai RPG.

Program doskonale sprawdza się podczas sesji online, między innymi prowadzonych przez Discorda. Wbudowane podpowiedzi oraz skróty zasad pozwalają ograniczyć konieczność korzystania z podręcznika podczas rozgrywki.

---

## Funkcje

- Tworzenie i edycja kart postaci
- Automatyczne obliczenia wybranych elementów mechaniki
- Wbudowane podpowiedzi i skróty zasad
- Intuicyjny interfejs użytkownika
- Zapisywanie i wczytywanie postaci
- Lekka aplikacja desktopowa dla systemu Windows
